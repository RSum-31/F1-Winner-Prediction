# F1-Winner-Prediction
🏎️ F1 Winner Prediction
Overview:-

This project is a Machine Learning-based Formula 1 Winner Prediction Web App that predicts the race winner based on historical race data. The application provides an interactive interface where users can select the Team, Grand Prix, and Year to predict the potential winner.
Dataset I used is from Kaggle 

This project was built to explore Machine Learning, Data Preprocessing, and Web App Deployment using Python.
 
Features:-
Interactive user interface
Predict Formula 1 race winner
Real-time prediction
Clean and simple design
Machine learning powered predictions

Machine Learning Workflow:-
Data Cleaning & Preprocessing
Label Encoding for categorical features
Feature Selection
Model Training using Random Forest
Prediction generation

Tech Stack:-
Python
Pandas
Scikit-learn
Streamlit
Machine Learning
 Input Parameters

Users can provide:
 Team
 Grand Prix
 Year

The model then predicts the likely winner of the race.

Project Goal:-

The goal of this project is to:

Practice Machine Learning concepts
Work with real-world data
Build an interactive web application
Deploy a ML model
